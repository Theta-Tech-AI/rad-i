# Setup
## Git

```
git clone https://github.com/Toth-Technology/rad-i && cd rad-i
```

## Docker

Install docker, then run:
### Linux:
```
docker run -it -p 8888:8888 -v $PWD:/root nathanhillyer/ml-base
```
### Windows:
```
docker run -it -p 8888:8888 -v ${PWD}:/root nathanhillyer/ml-base
```

Open in browser:
```
localhost:8888
```

Open `Juptyer/CoLlage.ipynb` to begin working.
